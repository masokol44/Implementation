📊 Матрица Эйзенхауэра – Multi‑Language Implementation
Тестовый репозиторий с реализацией консольной матрицы Эйзенхауэра на 7 языках программирования.
Каждая версия поддерживает управление задачами с разделением по важности и срочности, а также расширенные возможности для комфортной работы.

🧩 Что такое Матрица Эйзенхауэра?
Метод тайм-менеджмента, разработанный Дуайтом Эйзенхауэром, помогает расставить приоритеты, разделяя задачи по двум критериям:

Важность – влияет на достижение целей.

Срочность – требует немедленного выполнения.

На основе этих критериев задачи попадают в один из четырёх квадрантов:

Квадрант	Важность	Срочность	Рекомендация
1	✅ Да	✅ Да	Делать сейчас
2	✅ Да	❌ Нет	Запланировать
3	❌ Нет	✅ Да	Делегировать
4	❌ Нет	❌ Нет	Удалить / не делать
✨ Особенности
Интерактивное меню – выбор действий по номеру.

Задача содержит: id, название, описание, важность (да/нет), срочность (да/нет), статус выполнения (выполнена/нет), дата создания.

Автоматическое определение квадранта на основе важности и срочности.

Отметка выполнения – задача может быть помечена как выполненная.

Редактирование – изменение названия, описания, важности, срочности.

Удаление задач.

Фильтрация по квадрантам, по статусу выполнения.

Сортировка по дате, по id.

Сохранение / загрузка состояния в файл eisenhower_data.json (или эквивалент).

Цветной вывод (где поддерживается) для наглядного отображения квадрантов.

Статистика – количество задач в каждом квадранте и общее количество выполненных.
